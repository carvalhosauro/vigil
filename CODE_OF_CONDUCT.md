# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org),
version 2.1.

In short: be respectful, welcoming, and constructive. Harassment and
discrimination of any kind are not tolerated.

Unacceptable behavior may be reported to the maintainer at
**gustavo.carvalho@pigz.com.br**. All reports are handled confidentially.

The full text is available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
